# Project QUANTEX

Connecting technical problems to expert solutions instantly.

## Overview
An on-demand technical support platform that connects users with nearby technicians for fast home and office assistance.

## Features
- Technician booking
- Emergency support
- Home and office visits
- Responsive UI
- Fast issue categorization

## Tech Stack
- Replit
- TypeScript
- Node.js
- AI-Assisted Development
- Git & GitHub
